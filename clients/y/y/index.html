一、背景说明

机器人有限公司目前由两位股东组成：

岳学瑾（松竹亭公司）600,00060%出资 + 部分经营管理
聚仁会（常工）400,00040%出资 + 核心研发技术

公司当前以技术研发为核心，但市场推广、产品运营及商业化仍处起步阶段。为加快产品落地与市场开拓，公司拟邀请龚铁靖加入，负责产品与市场业务，作为股东及核心经营合伙人。

二、调整原则

为平衡出资与出力的贡献，建议采用“双层股权模型”：

总股权 = 资金股（60%） + 人力股（40%）

此模型可同时反映资金投入与经营贡献，兼顾公平与激励。

四、授予与约定机制

1️⃣ 授予方式（可选两种方式，经三方协商确定）

方案A：分期授予制（按业绩进度确认）

  • 16% 股权作为「人力股」，在 3 年内逐步确认：
    • 第1年：完成产品上市与首批客户签约 → 授予 8%
    • 第2年：完成销售/出货目标 → 累计至 12%
    • 第3年：实现盈利或完成融资目标 → 累计至 16%

方案B：一次性授予制（签约即生效）

  • 16% 股权于签署正式股东协议时一次性确认；
  • 可附带最低服务期（如满两年不得主动退出），以确保公司运营稳定性。

2️⃣ 退出与回购条款

  • 若龚铁靖提前退出公司，人力股部分由公司或其他股东按名义价值回购；
  • 若公司融资或股权变更，龚铁靖享有相应比例的优先认购权与防稀释权。

3️⃣ 决策与管理机制

  • 三方组成经营委员会（或董事会）；
  • 重大事项（增资、股权转让、公司清算等）须三方书面同意；
  • 日常经营由岳学瑾协调，龚铁靖主导市场与产品，常工主导研发技术。

五、实施意义

  1. 公平性：兼顾出资与贡献；
  2. 激励性：核心人员利益与公司增长绑定；
  3. 可扩展性：可延伸为未来员工持股或期权激励体系；
  4. 稳定性：保持岳学瑾控股、常工技术主导、龚铁靖市场主导的平衡格局。

六、结论与建议

调整后股权结构如下：

建议公司正式通过此股权结构方案，并在签署股东协议时确定采用**方案A（分期授予）或方案B（一次性授予）**的授予机制。

鉴于：

  1. 公司为推进产品商业化、市场拓展及运营增长，拟吸纳丙方作为核心经营合伙人；

三、授予标的与比例

经甲、乙双方一致同意，丙方获授予公司股权 16%（百分之十六），来源于公司现有股东的出让或公司增发。

四、授予方式（两种方案，三方选择其一执行）

方案A：分期授予制

  1. 丙方所获 16% 股权分三期授予，具体条件如下：
阶段授予比例授予条件
第一阶段8%丙方完成产品上市与首批客户签约；或完成公司规定的年度市场计划；
第二阶段4%丙方完成销售/出货目标，且公司市场营收达到约定标准；
第三阶段4%公司实现年度盈利或完成一轮融资；
  1. 每一阶段授予应经董事会确认，董事会通过后即生效；
  2. 若丙方在授予期内主动退出，公司有权按名义价值回购未生效部分股权。

方案B：一次性授予制

  1. 丙方的 16% 股权在本协议签署后一次性确认并登记；
  2. 为保证公司经营稳定，丙方承诺自签署日起 服务期不少于两年；
  3. 若丙方在两年内主动离职或终止合作，公司有权按原始登记价回购全部股权或部分股权；
  4. 超过两年后退出，丙方享有自由转让权（但须遵守股东优先购买权条款）。

五、权利与义务

  1. 丙方享有与股权比例对应的利润分配权、表决权及知情权；
  2. 丙方须积极履行市场拓展、品牌建设、商业合作、销售增长等职责；
  3. 丙方不得以任何形式从事与公司主营业务相竞争的项目；
  4. 公司须向丙方提供必要的业务支持、资源协调及数据透明度；
  5. 任何股权转让或增资，须经三方书面同意。

六、退出与回购条款

  1. 若丙方主动退出或严重违反公司规章制度，

    公司有权以原始出让价格或名义价值回购丙方全部或部分股权；

  2. 若公司经营方向、控制权发生变更，丙方享有防稀释权及优先认购权；
  3. 若公司被并购或上市，丙方持股按与其他股东相同条件结算。

七、保密与竞业禁止

  1. 丙方在任职期间及离职后两年内，不得泄露公司商业秘密；
  2. 未经公司书面许可，不得以直接或间接方式参与竞争性业务；
  3. 违约方需承担赔偿责任，金额不少于公司年度净利润的 20%。

八、争议解决

  1. 本协议适用中华人民共和国法律；
  2. 因本协议产生的争议，三方应先友好协商；
  3. 协商不成时，任何一方可向公司注册地有管辖权的人民法院提起诉讼。

九、其他条款

  1. 本协议自三方签字或盖章之日起生效；
  2. 本协议一式叁份，三方各执壹份，具有同等法律效力；
  3. 若后续需工商变更、签署股东协议等法律文件，本协议作为依据之一。

Energize Solutions Inc. 公司介绍

1. 公司概览 | Company Overview

Energize Solutions Inc. 总部位于美国德克萨斯州，由加拿大 AGI&M ASSETS INC. 全资控股，是一家专注于 微电网控制系统（EMS）与工商业储能解决方案 的科技公司。公司以自研的 EnergizeOS™ 能源管理操作系统 为核心，为全球客户提供从系统设计、控制柜集成到软件策略的全栈式解决方案。

我们的团队长期服务于 EPC 总包商、能源开发商、OEM 厂商及大型工业终端客户，在美国、加拿大、欧洲及亚太地区均有项目落地经验。Energize Solutions 致力于通过技术创新与工程标准化，推动全球能源系统的智能化、可再生化与可调度化。

2. 核心业务与产品体系 | Core Business & Product Portfolio

Energize Solutions 的核心产品与服务分为三大体系：

🔹 1. EnergizeOS™ 软件系统

  • 微电网能源管理系统（EMS）
  • 策略模块（Strategy Modules）：需量管理、分时电价优化、并离网切换、新能源消纳、柴油机协调等
  • 云端与本地协同控制架构（Edge + SaaS）
  • 智能日志、报警、权限与远程策略推送机制

🔹 2. 控制柜系列（ECP / AIPP）

  • ECP – EnergizeOS™ EMS Control Panel

    工业级控制柜，内置 EnergizeOS™ 系统，可实现储能系统与电网、柴油机、光伏的多源协同。

  • AIPP – Anti-Islanding Protection Panel

    防孤岛保护成套柜，符合 IEEE 1547 / UL 1741 SB / Rule 21 等标准，具备独立 UPS 供电与合闸互锁判据（C1–C7）逻辑。

🔹 3. 系统设计与工程服务

  • 微电网架构设计与接入方案
  • FAT 工厂测试、联调测试与远程诊断
  • 控制逻辑设计、点表配置、现场调试指导
  • UL / IEEE / CSA 等国际标准合规咨询

3. 技术架构与研发方向 | Technical Architecture & R&D Focus

EnergizeOS™ 采用 五层架构设计：

核心引擎层 → 功能模块层 → 数据接口层 → 设备配置库 → UI & API 层。

系统基于 Linux（Ubuntu / Debian）开发，支持 x86 与 ARM 架构，控制逻辑采用 Python，前端界面采用 Vue.js 构建，并通过 Docker 实现容器化部署。

未来研发重点包括：

  • AI 负载预测与主动策略优化模块；
  • 能源调度仿真引擎（Simulation Engine）；
  • 分布式多节点控制 与 策略沙盒机制；
  • 远程 OTA 策略与安全更新体系。

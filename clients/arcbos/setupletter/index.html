二、合伙企业治理设计（合伙协议核心）

原则：LP出资换取经济权益，不取得对运营公司经营决策的干预权；GP承担管理与执行责任，权责必须边界清晰。

2.1 GP权限：全权执行事务（建议条款要点）

  • *目的：**对外统一口径，避免“LP参与经营”引发连带风险与内耗。

2.2 LP权利边界：不参与经营、仅享经济权益（建议条款要点）

LP权利建议限定为：

  • 查阅合伙企业年度财务报表、审计报告（可设信息披露范围与频率）；

  • 依约获得分配、回购、退出结算；
  • 对GP严重违约时的救济路径（如更换GP的严格条件、司法救济等）。

LP禁止事项建议明确：

  • 不得干预GP执行事务；
  • 不得要求合伙企业直接干预运营公司人事/财务/采购/研发等经营决策；
  • 不得擅自对外披露商业秘密或对外作出投资承诺；
  • 不得以合伙份额质押、代持、信托等变相转让方式规避限制。

2.3 “重大事项”决策机制：避免过度锁死，采用“保留事项清单”

你提到“重大事项需GP决议”。为了投融资可接受性，建议采用：

  • 日常事务：GP单独决策
  • 保留事项：LP按约定比例表决（但比例与范围要控）

建议保留事项（示例，律师可微调）：

  1. 合伙企业解散、清算；
  2. 合伙协议修改中涉及LP经济权益的核心条款（分配、回购、退出、稀释口径）；
  3. 新增重大对外担保/借款（以合伙企业名义）；
  4. 向外转让运营公司股权（合伙企业持有的30%整体转让）或设定重大权利负担；
  5. 重大关联交易（GP或其关联方与合伙企业之间）。

关键：把“保留事项”控制在合伙企业层面，不要把运营公司经营事项都变成LP表决，否则平台会变成权力平台、融资很难。

三、退出机制（LP最关心，必须写死“规则”但不能写死“结果”）

3.1 退出类型与处理逻辑（建议分四类）

  1. 到期退出：约定锁定期满后，LP可按规则申请退出；
  2. 提前退出（无过错）：锁定期内退出，适用折价回购/延期支付；
  3. 违约退出（有过错）：违反保密/竞业/挖人挖客户/恶意阻碍等，适用惩罚性回购+违约金；
  4. 强制退出：触发法定或约定事由（失信、刑事风险、重大声誉风险等）。

3.2 锁定期（Lock-up）

建议：24–36个月作为基础锁定期（可与你“3年不能退出”的想法一致，但建议用法律语言表达为“锁定期内不得退伙/不得转让”，并设例外）。

例外（可选）：

  • 经GP同意且不影响融资/经营；
  • 投资人更替/家庭重大变故等经GP批准的特殊情形。

3.3 退出结算与支付安排（避免现金挤兑）

建议写入：

  • 退出回购价确定后，允许分期支付（12–24个月）
  • 可设“支付前提”：不影响公司现金流、融资进程、重大项目交付等
  • 允许以“股权置换/新投资人承接”替代现金回购（如发生融资）

四、分红与收益分配规则（让投资人敢投、也避免扯皮）

4.1 分配顺序建议（常见且可接受）

建议采用如下顺序（律师可据你商业策略调整）：

  1. 缴纳税费及必要支出；
  2. 预留风险准备金/运营资金（比例或金额上限写明）；
  3. 先行偿付合伙企业对外债务（如有）；
  4. 向LP分配（可约定优先回收本金/或按份额分配）；
  5. GP业绩报酬（如你希望设置“carry”，可在达到某一IRR/倍数后触发）；
  6. 余额按份额分配。

4.2 分配频率与触发条件

  • 分配频率：建议年度/半年度；
  • 触发条件：以合伙企业收到来自运营公司的分红/退出款/股权转让款为前提；
  • 信息披露：分配前提供报表/审计摘要，降低信任成本。

五、回购机制（必须“可执行”，不要写情绪化条款）

你要的“可强制回购”“明确回购公式”，建议这样做：

5.1 回购主体

回购主体可设计三选一（律师判断合规与税务）：

  1. 合伙企业回购LP份额（最常见）；
  2. GP指定第三方承接（避免合伙企业现金压力）；
  3. 新投资人承接（融资时常用）。

不建议直接让“运营公司回购LP份额”，税务与公司法限制更多，且会把平台与公司财务绑死。

5.2 回购触发事件

  • 无过错退出（到期/经同意提前退出）；
  • 违约事件（保密、竞业、挖人挖客户、对外不当承诺、损害声誉等）；
  • 司法/信用风险（失信、冻结、被执行等）；
  • 触发跟投义务但拒绝履行（见第七部分）。

5.3 回购价格公式（给律师可落地口径）

建议设“三轨制”，兼顾公平与可执行：

轨道A：到期正常退出（公平定价）

  • 约定“公允价值”机制：以最近一轮融资估值为基准；若无融资，则以审计净资产×系数/或第三方评估；
  • 设置上下限，避免争议拖死。

轨道B：锁定期内提前退出（折价定价）

  • 按出资额×（1 - 折价率）
  • 或按公允价值×折价系数（如0.6–0.8）
  • 并允许分期支付。

轨道C：违约退出（惩罚性回购）

  • 按出资额×更低系数（如0.2–0.5）或按名义价；
  • 另设违约金（固定金额或按投资额比例），并明确可与回购款抵扣。

关键：不要写“净身出户”。写“违约回购价=名义价/大幅折价 + 违约金”，法院更容易支持。

六、表决权与“平台不变权力平台”的硬约束

你提出“这些份额是否有表决权”。建议明确如下：

6.1 LP在合伙企业内的表决权

  • LP对合伙企业“保留事项”享有表决权；
  • 其他事项无表决权或仅建议权。

6.2 LP对运营公司层面的表决权

必须写清：

  • LP不直接持有运营公司股权
  • LP不享有运营公司股东权（表决/提案/任免等）
  • 合伙企业作为股东的权利由GP统一行使

并进一步约定：

  • GP对运营公司表决可不经LP同意（除非涉及“保留事项中与LP经济权益直接相关”的少数情形）

七、未来融资与稀释机制（提前写清，避免融资时撕裂）

你提出“未来融资是否稀释”，建议采用“统一稀释规则”：

7.1 稀释原则

  • 运营公司发生增资融资，所有股东按持股比例稀释（含合伙企业的30%）；
  • 合伙企业内部LP份额对应的“穿透比例”随之同比例稀释（自动发生，无需单独协议）；
  • 该稀释为“股权融资常规后果”，LP不得以此主张违约或回购。

7.2 跟投义务（Pro-rata / Pay-to-play）

你要求“明确跟投义务”，建议：

  • A轮及以后融资，LP享有按份额比例的优先认购权；
  • 但可设置“跟投义务”：在特定融资轮次，LP必须按比例跟投，否则触发惩罚（例如：丧失部分优先权/被动稀释加速/份额被强制转让给跟投方）。

中国实务里“强制跟投”条款要写得谨慎：建议做成“权利丧失/份额调整”而非单纯“必须掏钱，否则违法”，由律师把握尺度。

八、转让限制（禁止单独转让，防止平台被拆穿）

你要求“禁止单独转让份额”，建议条款组合拳：

8.1 锁定期内禁止转让

  • 锁定期内不得转让/质押/设信托/代持等变相转让。

8.2 锁定期后转让仍需满足“三道门”

  1. GP同意（核心）；
  2. 优先购买权：合伙企业/其他LP/GP按顺序优先；
  3. 合格受让人：不得转让给竞争对手/失信人员/敏感主体。

8.3 禁止拆分转让

  • LP不得拆分份额转让给多个受让人（除非GP批准），避免股东人数失控与复杂化。

九、Vesting机制（合伙企业层面，解决“贡献兑现”问题）

你提出“合伙企业层面Vesting”，建议用于两类人：

  1. 以“联合创始人/关键合伙人”身份进入平台但贡献未完全兑现的人；
  2. 未来引入的“资源型LP/渠道型LP”但承诺要完成业绩的人。

9.1 Vesting的实现方式（建议）

  • 不是直接写“份额归属”，而是：
    • 先授予“受限制份额/附条件份额”
    • 未达成里程碑 → GP有权按名义价回购未归属部分
    • 达成里程碑 → 份额逐期解锁

9.2 里程碑建议写法

  • 用可量化成果：签约金额/毛利/交付节点/专利/关键系统交付等；
  • 避免“努力、支持、协助”这种不可执行词。

十、Drag-along / Tag-along（并购退出时必须）

你要求 Drag-along，建议：

10.1 Drag-along（强制随售）

当满足以下条件时，LP必须随售：

  • 运营公司整体出售/控制权变更
  • 或合伙企业转让其持有的运营公司股权
  • 达到约定价格/估值门槛（避免低价卖出）

10.2 Tag-along（跟随出售）

若创始人/合伙企业对外出售股权，LP可按比例跟随出售其经济权益（或由合伙企业内部实现对价分配）。

十一、保密、竞业、禁止挖人挖客户（必须可执行、可举证）

建议：

  • 保密义务：范围、期限、违约金；
  • 竞业限制：期限与地域合理（一般不超过2年），并考虑是否需要补偿（劳动关系更关键）；
  • 禁止挖人挖客户：更可执行，建议重点写；
  • 违约责任：可与回购款抵扣、并可申请禁令/行为保全（律师评估可行性）。

十二、文件闭环与落地清单（请律师交付）

请贵所据此建议形成以下文本并保证一致性：

  1. 《有限合伙协议（最终版）》
  2. 《GP公司章程/设立文件》及GP授权体系
  3. 《运营公司章程修订版》（与平台治理一致）
  4. 《股东会决议/合伙人会议决议》模板（资本公积、出资溢价、对外融资、股权转让等）
  5. 《入伙协议/LP认购协议》模板（含KYC、反洗钱、信息披露确认）
  6. 《退出与回购协议》模板（或合伙协议内嵌条款）
  7. 《保密与禁止挖人挖客户协议》模板
  8. 《融资时投资人进出机制条款包》（优先认购、稀释、跟投、交割与信息披露）

十三、律师需特别出具“合规可执行性意见”的问题清单

请贵所重点核验并给出意见：

  1. 溢价出资计入资本公积的工商流程与文件路径；
  2. 合伙企业份额回购/转让限制条款在合伙企业法框架下的可执行性边界；
  3. “跟投义务/惩罚机制”的合法表达方式（避免被认定无效）；
  4. 违约回购价与违约金的司法可支持区间（显失公平风险控制）；
  5. Drag-along 在有限合伙内部如何实现对价分配与强制执行；
  6. 信息披露边界（既满足LP知情，又避免泄密与干预经营）；
  7. 税务与会计处理口径（回购、分配、资本公积、份额转让税负）。

结语（你可放在函末）

本建议函旨在将委托人既定控制权与持股平台架构落地为可执行、可融资、可审计的条款体系。请贵所据此完善为正式法律文件，并在确保合法合规前提下，优先实现以下三项目标：

  1. 创始人持续保持对运营公司的控制权与决策权；
  2. LP经济权益清晰、退出机制可预期，减少纠纷；
  3. 结构对后续融资友好、可延续、可升级。

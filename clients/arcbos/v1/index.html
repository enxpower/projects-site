ARCBOS 全电动-全自治-无人重载除雪机器人-结构设计

冻结级 结构设计基线 Baseline v1.0

一台重载级、全电动、自主运行的除雪机器人。
紧凑型工业尺度，整体占地尺寸接近大型商用履带式除雪机。
该设备是工业机器人，而不是车辆。

【核心身份约束】
这是一个完全无人化的工业机器人系统。
在任何设计层面，都不允许承载、容纳或服务人类。
外观与结构必须明确避免任何“可驾驶设备”或“有人操作机械”的暗示。

【绝对禁止的元素】
- 任何形式的驾驶舱、座舱、操作舱
- 挡风玻璃、玻璃、窗户、透明面板
- 顶棚、车顶结构、雨棚
- 雨刷、后视镜
- 方向盘、座椅、把手、踏板
- 类车辆比例或车身语言
- 任何用于传感器或灯具的立杆、立柱、桅杆
- 科幻化造型、消费级产品风格、玩具感设计

【整体造型语言】
- 装甲化工业机器人平台
- 开放式结构或装甲钢壳式结构
- 平直、折线化或多面焊接钢板
- 清晰可见的螺栓、紧固件、焊缝、加强筋
- 加强型边角与抗冲击结构
- 可拆卸、螺栓固定的牺牲式耐磨部件
- 不存在任何足以容纳人体的封闭空间
- 明确呈现为“机器人/机械系统”，而非车辆

【工业设计哲学】
- 功能优先，零装饰
- 机械真实感：结构与受力路径可被直观看出
- 设计目标是承受冰块、路沿、硬雪、杂物的反复冲击
- 所有部件看起来都可维护、可更换
- 商用级工业设备，而非消费级产品

【大致尺寸比例】
- 宽度约 1.0 米
- 长度约 1.5 米
- 高度约 1.2 米
- 紧凑、密实的工业比例
- 低重心设计
- 短前后悬，无高耸上部结构

【底盘与材料】
- 重型焊接钢结构底盘
- 厚实钢板，清晰可见材料厚度
- 外露工业级紧固件
- 围绕除雪系统的加强型副车架
- 工业黄色 + 深灰色配色
  灵感来源于卡特彼勒（Caterpillar）与约翰迪尔（John Deere）
- 哑光或半哑光工业涂层
- 禁止高光、装饰性表面处理

【前端除雪系统】
- 商用级重载双级除雪系统
- 前置螺旋绞龙直接喂入中央叶轮
- 绞龙与叶轮处于同一连续壳体内
- 除雪流向在视觉与物理上必须真实合理
- 叶轮将雪垂直向上抛出
- 抛雪筒直接安装在叶轮壳体正上方
- 抛雪筒仅左右旋转，底座固定
- 抛雪筒为厚钢板结构，不得使用塑料外观
- 禁止侧向抛雪或不合理的气流路径

【切削边与耐磨件】
- 清晰可见、可更换的金属切削刀刃
- 螺栓固定的牺牲式切削边
- 硬化钢耐磨滑靴
- 耐磨件与结构件在视觉上有明确区分
- 重载商用级钢结构外观

【履带行走系统】
- 厚实的工业级橡胶履带
- 紧凑型履带驱动系统
- 相对于高度具有较宽的履带站距
- 针对积雪、结冰、不平整路面优化
- 不得出现轮胎
- 工业级驱动轮与导向轮部分外露

【动力与系统属性】
- 全电动驱动系统
- 电池供电的工业设备
- 不得出现排气口或暗示内燃机的进气结构
- 密封式电气舱
- 可隐约看到高压电安全警示标识

【传感器与自主系统】
- 激光雷达集成于低矮装甲化传感器舱内
- 工业相机嵌入装甲式传感器模块
- 传感器外置但与结构高度一体化
- 禁止玻璃罩、透明穹顶
- 禁止任何传感器立杆或高耸支架
- 传感器布局支持 360 度环境感知
- 所有感知硬件为工业级、抗冲击、抗低温设计

【安全与照明系统】
- 低矮圆柱形工业琥珀色 LED 警示灯
- 警示灯完全嵌入顶部装甲结构
- 禁止立杆、支架、外伸结构
- 短而宽、重载型警示灯
  风格接近机场或重型施工设备
- 前向高亮白色工业工作灯
- 后向琥珀色警示灯
- 侧向嵌入式示廓灯
- 所有灯具均为密封、防冲击、抗振动结构
- 适用于无人化、公共或半公共环境运行
- 符合商用设备可视与安全预期

【运行特征】
- 自主优先设计（Autonomous-first）
- 无任何明显人工操作界面
- 设计用于夜间、无人值守连续运行
- 工业可靠性高于外观美感
- 安静、电动、目的性极强的存在感

【场景与环境】
- 机器人正在商业停车场内执行除雪作业
- 暴雪环境，雪量大、雪花密集
- 雪流清晰可见地通过绞龙、叶轮与抛雪筒
- 夜间或低照度环境
- 工作灯与警示灯照亮飞雪颗粒
- 警示灯处于闪烁状态
- 场景中无任何人类
- 无近距离交互的车辆
- 电影级但真实的工业摄影风格
- 高真实度、物理逻辑合理
- 专业级商用工业设备气质

结构工程师任务树（Work Breakdown Tree / WBS）

ARCBOS SnowBot（全电动无人重载除雪机器人）
└─ 0. 全局硬约束（所有结构设计必须同时满足）
   ├─ 身份约束：工业机器人，不是车辆（禁止任何车辆语义）
   ├─ 绝对禁止：驾驶舱/座舱/操作舱、玻璃/窗、雨棚/车顶、方向盘/座椅/把手、雨刷/后视镜
   ├─ 禁止立杆/桅杆：任何传感器与灯具不得使用立柱、外伸支架
   ├─ 外形语言：装甲化工业平台；焊接钢板；可见焊缝、螺栓、加强筋；牺牲式耐磨件可更换
   ├─ 尺寸边界：W≈1.0m / L≈1.5m / H≈1.2m；低重心；短前后悬；无高耸上部结构
   ├─ 商用级：抗冲击/抗低温/抗振；可维护可更换；结构真实感（受力路径可读）
   └─ 外观限制：哑光/半哑光工业涂层；工业黄+深灰（CAT/John Deere 气质）；拒绝玩具感/科幻感/装饰面

└─ 1. 总体结构架构（结构总工牵头）
   ├─ 1.1 总体布置与结构分区（Frame + Modules）
   │  ├─ 目标：形成“开放式结构/装甲壳”但无人体容积；可快速拆装维护
   │  ├─ 关键点：结构受力路径清晰；雪流通道不被结构干扰；传感器/灯具必须低矮内嵌
   │  └─ 交付物：总体布置图（2D/3D）、模块安装面定义、质量/重心预估、维修开盖路径图
   ├─ 1.2 强度/刚度/振动基础指标（冬季工况）
   │  ├─ 工况：冰块冲击、路沿磕碰、硬雪反力、履带振动、叶轮高转速激振
   │  └─ 交付物：设计载荷表、结构安全系数假设、初版FEA计划（静强度+模态）
   └─ 1.3 模块化与可维护性策略
      ├─ 目标：关键部件“快拆+标准紧固件+可替换耐磨件”
      └─ 交付物：模块拆装序列、备件/牺牲件清单、检修空间与工具可达性评估

└─ 2. 主承载底盘与焊接钢车架（Chassis / Main Frame）
   ├─ 2.1 焊接主框架（重载钢结构）
   │  ├─ 要求：厚钢板、可见焊缝与加强筋、结构“机械真实感”
   │  ├─ 防跑偏：不能做成“汽车底盘+外壳”的车辆套路；必须像工业机器人底座
   │  └─ 交付物：焊接总成图、焊缝等级与工艺建议、吊装/搬运点设计
   ├─ 2.2 加强副车架（Snowblower Module Subframe）
   │  ├─ 目标：前端除雪模块与主框架之间形成抗冲击隔离/加强结构
   │  └─ 交付物：安装基准面、定位销/螺栓组、抗冲击加强件设计
   └─ 2.3 密封电气舱结构（非透明/非玻璃）
      ├─ 要求：密封、防雪水侵入、抗振；可隐约呈现高压警示标识
      └─ 交付物：电气舱外壳图、密封方案、检修门与锁扣方案

└─ 3. 履带底盘结构（Tracked Undercarriage）
   ├─ 3.1 履带模块与安装界面
   │  ├─ 要求：厚工业橡胶履带；宽站距；低重心；不出现轮胎语义
   │  └─ 交付物：履带模块安装图、受力路径说明、快速更换方案
   ├─ 3.2 驱动轮/导向轮防护与半外露策略
   │  ├─ 要求：工业驱动轮、导向轮可部分外露（真实机械感），但要有防雪护罩
   │  └─ 交付物：护罩结构图、排雪/排泥通道、维护窗口
   └─ 3.3 冲击与振动隔离（结构层面）
      ├─ 目标：控制传到传感器舱、灯具与电气舱的振动
      └─ 交付物：隔振点位建议、橡胶/弹性支座策略（只给结构接口，不限定供应商）

└─ 4. 前端双级除雪系统结构（Auger + Impeller + Housing）
   ├─ 4.1 连续壳体（Aligned Housing）
   │  ├─ 关键约束：绞龙→叶轮的雪流必须连续、可视化正确；不能出现不合理风道
   │  └─ 交付物：壳体3D、雪流通道剖面、维修开口与防卡堵设计
   ├─ 4.2 抛雪筒（Chute）与旋转机构的结构接口
   │  ├─ 硬约束：抛雪筒底座固定，仅左右旋转；抛雪筒厚钢板外观（非塑料）
   │  └─ 交付物：旋转支承结构、限位设计、雪密封与防冻结策略
   └─ 4.3 前端抗冲击与防卷入防护结构
      ├─ 目标：石子/冰块冲击、边沿磕碰；同时防止杂物卷入
      └─ 交付物：前沿护板、格栅/防护罩方案、冲击损伤可替换件

└─ 5. 切削边与牺牲耐磨件系统（Cutting Edge & Wear Parts）
   ├─ 5.1 可更换金属切削刀刃（Bolted-on）
   │  ├─ 关键：必须“肉眼可见”是可替换件；螺栓固定；与结构件明显区分
   │  └─ 交付物：刀刃零件图、孔位标准、替换流程与扭矩建议
   ├─ 5.2 耐磨滑靴/耐磨板（Wear Shoes/Skids）
   │  ├─ 目标：适配地面不平、路沿；牺牲件优先磨损
   │  └─ 交付物：耐磨件布置、可调高度策略（结构接口）
   └─ 5.3 牺牲式装甲护板（Sacrificial Armor Plates）
      ├─ 目标：碰撞先坏外层，保护主车架；体现“装甲工业机器人”语言
      └─ 交付物：护板分区图、快拆紧固件、备件编号规则建议

└─ 6. 装甲外壳与开放式结构（Armored Shell / Open-frame）
   ├─ 6.1 装甲钢板分块与折线语言
   │  ├─ 要求：平直/多面焊接钢板；可见焊缝/螺栓；拒绝流线车壳
   │  └─ 交付物：外壳分块方案、折弯/焊接工艺建议、装配公差策略
   ├─ 6.2 “无人化”外观约束核查
   │  ├─ 检查点：任何看起来像“人可以进去/坐进去/操作”的空间都必须消除
   │  └─ 交付物：无人化核查清单（结构版）、整改清单与对策
   └─ 6.3 维护开口、检修门与锁扣系统
      ├─ 目标：可维护，但不形成“门/驾驶舱”错觉；门尺寸不能暗示人体进入
      └─ 交付物：检修门方案、锁扣类型建议、密封与防雪水策略

└─ 7. 传感器舱与安装结构（Sensors Integration, NO MAST）
   ├─ 7.1 LiDAR 低矮装甲舱（Top Integrated）
   │  ├─ 硬约束：低矮、内嵌、装甲化；绝对禁止透明罩与玻璃穹顶
   │  └─ 交付物：LiDAR舱结构件、视场遮挡评估（结构侧）、防撞边缘设计
   ├─ 7.2 工业相机装甲模块（Flush / Embedded）
   │  ├─ 要求：嵌入式、抗冲击、防雪冰；不外伸、不立杆
   │  └─ 交付物：相机模块安装座、保护窗（非玻璃语义，工程防护件）、更换流程
   └─ 7.3 360°视野的结构避让与走线保护
      ├─ 目标：结构不遮挡视野；走线全程装甲/护套；避免被冰雪拉扯
      └─ 交付物：视野遮挡地图、走线槽/护管方案、固定点位图

└─ 8. 安全照明与信号灯结构（Lighting Integration, LOW-PROFILE）
   ├─ 8.1 顶部低矮圆柱琥珀警示灯（无立杆）
   │  ├─ 硬约束：必须“嵌入顶部装甲结构”，短而宽；绝对禁止立杆式旋转灯
   │  └─ 交付物：灯座结构、抗振与防撞结构、维护更换路径
   ├─ 8.2 前向白色工作灯（工业级）
   │  ├─ 要求：嵌入式/装甲保护式；密封抗冲击；夜间照雪颗粒真实可见
   │  └─ 交付物：灯具安装位、散热与防雪堵策略、线束固定点
   ├─ 8.3 后向琥珀灯与侧向示廓灯（Flush-mounted）
   │  ├─ 要求：侧灯必须“齐平嵌入面板”，避免突出挂碰
   │  └─ 交付物：嵌入开孔标准、密封垫圈结构、抗冲击边框
   └─ 8.4 灯具防护等级与结构验证
      ├─ 目标：封闭、防水、防雪、抗振；商业设备安全可视性
      └─ 交付物：结构验证条目（IP/振动/冲击）与测试夹具需求

└─ 9. 防雪、防冰、排水与可用性结构细节（Winterization by Structure）
   ├─ 9.1 排雪/排水通道（避免积雪结冰）
   │  ├─ 目标：所有水平面避免积雪堆积；形成排水坡度与导流槽
   │  └─ 交付物：导流结构、排水孔位、易结冰风险点清单
   ├─ 9.2 防夹雪缝隙与防冻卡结构
   │  ├─ 目标：旋转件/铰链/锁扣避免冻死；关键机构留合理间隙与防护
   │  └─ 交付物：间隙规范建议、关键机构防护罩设计
   └─ 9.3 冲击区“牺牲件优先”的维修策略
      ├─ 目标：常碰撞区域先损坏可替换件，不伤主结构
      └─ 交付物：冲击热区地图、牺牲件库存建议、现场更换工装建议

└─ 10. 结构交付物清单（给结构团队的最终验收）
   ├─ 10.1 冻结版3D（装配）+ 2D（关键零件图）
   ├─ 10.2 焊接总成与工艺建议（焊缝/定位/变形控制）
   ├─ 10.3 维护拆装序列（含牺牲件更换流程）
   ├─ 10.4 结构载荷与初版FEA结果（静强度+模态+关键冲击点）
   ├─ 10.5 无人化约束核查报告（车辆化风险清单=0）
   └─ 10.6 冬季结构风险清单（结冰、积雪、卡滞、密封失效）与对策

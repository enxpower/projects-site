附录：创始人控制权保护条款

（Founder Control Protection）

一、创始人不可罢免保护

在公司章程中建议明确：

创始人龚铁靖

担任公司执行董事及总经理

除非本人主动辞任

不得被股东会单方面罢免

若需更换：

必须经

创始人本人同意

二、融资清算优先权（Liquidation Preference）

适用于未来机构投资。

原则：

投资人享有

1倍清算优先权

即：

公司出售或清算时

投资人

优先收回投资本金

不建议：

2倍或更高优先权

以避免：

创始人收益被严重压缩

三、投资人联合控制风险防火墙

在股东协议中明确：

任何投资人

不得联合其他投资人

通过股东协议

改变公司控制权结构

即：

创始人控制权

不可被投资人联盟剥夺

四、融资优先认购权

未来融资时：

现有股东

享有优先认购权

但：

创始人拥有最终融资决策权

以确保：

公司融资灵活性

五、法律文件一致性要求

律师起草文件时需确保：

公司章程
股东协议
投资协议

三者：

保持一致

并确保：

控制权条款

写入公司章程

投资条款：

写入股东协议

以确保：

法律有效性
与工商备案可行性

Tag-along（共同出售权）——写入股东协议：
│
└─ 控股股东出售股权 → 其他股东可同条件一起卖

Drag-along（强制随售权）——写入股东协议：
│
└─ 创始人或≥51%股东同意整体出售公司
   → 其他股东必须配合签署所有文件
   → 不得以优先购买权/拖延方式搅局


┌──────────────────────────────────────────────┐
│  G. 融资层（反稀释 + 优先认购 + 清算优先）       │

反稀释（Anti-dilution）——写入投资协议：
│
└─ 下轮估值低于上一轮 → 加权平均反稀释

优先认购权（Pre-emptive Right）：
│
└─ 现有股东按比例优先认购（但融资决策权归创始人）

清算优先权（Liquidation Preference）：
│
└─ 建议 1x（本金优先返还），避免过高倍数挤压创始人收益

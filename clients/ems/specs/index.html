工程级微电网能源管理系统（EMS）

一、产品是什么

EnergizeOS™ 是一套工程级微电网能源管理系统（EMS），用于对包含电网、电池储能（BESS）、光伏（PV）、柴油机（DG）及负荷的微电网系统进行运行态裁决、控制授权、多能源协调与审计记录。

它不是监控软件，也不是算法平台，而是位于继电保护之上的监督控制与决策系统，负责决定：

  • 系统是否并网 / 离网 / 再并网
  • 主断路器是否允许合闸
  • 多能源在不同运行态下如何协同
  • 在异常情况下系统如何退回安全态

二、产品解决什么问题

在真实工程项目中，微电网面临的核心问题不是“设备不够先进”，而是：

  • 并离网切换逻辑混乱
  • 合闸控制权分散、不可审计
  • 多能源抢控制权、系统不稳定
  • 出事后无法回溯责任链
  • 项目越做越像定制工程，无法复制

EnergizeOS™ 的目标，是把这些问题工程化解决。

三、核心能力（产品级）

1. 并网 / 离网 / 再并网裁决引擎

  • 显式状态机
  • 明确切换条件、顺序与失败回退
  • 可审计、可回放

2. 合闸唯一授权机制

  • EMS 为 PCC 合闸的唯一合法授权者
  • 多条件互锁（状态 / 判据 / 权限 / 保护）
  • 禁止任何绕过 EMS 的自动合闸路径

3. 多能源协调调度

  • BESS / PV / DG / Grid 角色分工明确
  • 并网态、离网态差异化控制
  • 可解释、规则优先的调度策略

4. 关键负荷保障

  • 负荷分级
  • 离网切换与恢复顺序控制
  • 与 UPS、ATS、DR 系统协同

5. 审计级日志系统

  • 所有状态变化、控制命令、执行结果可追溯
  • 满足工程复盘、保险、仲裁需求

6. 本地生存能力

  • 离线运行
  • 本地 HMI
  • 云、通信失效不影响核心裁决

四、产品架构原则

Global Core + Regional Add-ons

  • Global Core（全球统一内核）
    • 并离网状态机
    • 合闸授权模型
    • 多能源协调
    • 日志与安全
  • Regional / Functional Add-ons
    • 电价与需量规则
    • 柴油节省策略
    • 需求响应
    • AI 优化模块

Core 不分国家、不分项目、不分规模

五、适用客户

1. 工商业微电网业主

  • 工厂、园区、数据中心
  • 关注：电费、停电风险、可复制性

2. EPC / 系统集成商

  • 需要标准化 EMS
  • 降低项目定制成本
  • 减少工程风险

3. 储能与设备 OEM

  • 为硬件提供“工程级大脑”
  • 提升系统级竞争力

六、典型应用场景

  • 并网 + 储能削峰填谷
  • 带孤岛能力的工商业微电网
  • 光储柴混合系统
  • 高可靠性关键负荷供电系统

七、产品交付形态

  • 本地部署（工业控制柜 / IPC）
  • 支持 x86 / ARM
  • 标准通信接口（Modbus / IEC / DNP3）
  • 可远程升级、可版本回滚

八、商业模式

1. 基础授权（必选）

  • EnergizeOS™ Global Core
  • 按设备 / 系统授权（License）

2. 功能订阅（可选）

  • 柴油优化
  • 电价优化
  • AI 负载预测
  • 按月 / 年订阅（SaaS）

九、开发与交付节奏（示意）

  • Phase 1：Core 稳定版（工程可交付）
  • Phase 2：区域插件与行业方案
  • Phase 3：AI 与规模化平台能力

十、产品定位总结（一句话）

EnergizeOS™ 不是“管能耗的 EMS”，而是“为微电网提供运行裁决、控制权与责任边界的工程级操作系统”。

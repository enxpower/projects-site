IB Rack Shuffle Kit（W80S0）

中文版

这两天我重新系统性梳理了一遍 Microsoft 的 IB Rack / Data Center 项目，结合现有架构和 BOM，想先和你对齐一个务实、可快速落地的推进思路。

从执行角度看，如果一开始铺得太散，容易拉长验证周期、分散资源。我们的建议是：先从成功率最高、最容易形成长期合作的切入点开始，把第一步走稳、走实。

第一阶段，我们建议优先聚焦两条线：

1）IB Rack Shuffle Kit（MPO8 光纤线束系统）

这是 Rack 级关键配套，核心价值在一致性、极性管理、测试和交付可靠性，一旦验证通过，后续会随每一柜持续重复使用。

2）电源线（C14–C15）等标准化配套件

技术和认证路径清晰，单柜数量稳定，适合在前期同步推进，加快整体落地节奏。

这样推进，一方面可以先把核心能力和交付可信度跑通，另一方面也能在不冒进的前提下，尽快形成实质性进展；同时也更符合 Microsoft / Hyve 在二级供应商导入时“先稳、再放量”的节奏。

如果你这边认可这个方向，我们就可以按这个思路一起往前推进，过程中需要我补充的材料或说明，也可以随时对齐。

English Version

I’ve taken some time to re-review the Microsoft IB Rack / Data Center project in a structured way, and I’d like to align with you on a practical and execution-focused approach that we can move forward with together.

From an execution perspective, covering too many components at the outset tends to slow validation and dilute focus. Our view is to start with the areas that have the highest probability of success and long-term repeat value, and make sure the first step is solid.

For Phase 1, we suggest focusing on two priority tracks:

1) IB Rack Shuffle Kit (MPO8 fiber cabling system)

This is a rack-level integration item where the value lies in consistency, polarity management, testing, and delivery reliability. Once qualified, it typically becomes a repeat item across rack deployments.

2) Power cords (C14–C15) and other standardized accessories

These have clear technical and certification paths, stable quantities per rack, and are well suited to move in parallel to accelerate early progress.

This approach allows us to establish technical credibility and delivery confidence first, while generating real progress without unnecessary risk. It also aligns well with how Microsoft / Hyve typically onboard Tier-2 suppliers — stability first, scale later.

If this direction makes sense to you, we can proceed along this path and align on next steps. I’m happy to provide any additional materials or clarification as needed.

A2. 技术规格（不可协商项）

A2-1 光纤/连接器

  • Fiber type：OS2 Single Mode
  • Connector：MPO8
  • Polarity：Type A ONLY
  • Gender：按 BOM（Female/Male）
  • Ferrule endface：UPC（默认）
  • Cable jacket rating：CPR / Riser（按 BOM）
  • Operating temp：-10°C ~ +60°C（或厂商标准，需声明）

硬要求：供应商必须能“书面解释” Type A 极性规则，并提供自检流程（见询价问题）。

A2-2 性能与测试（必须 100%）

A3. 交付要求（包装与文件）

  4. 端面检测标准声明（IEC 61300-3-35 等）

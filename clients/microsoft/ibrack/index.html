Unified Naming Logic (For Internal / MS Communication)

  • Tier-1: Architecture owner / MS-direct vendor
  • Tier-2 (Secondary Source): Execution manufacturer under Tier-1 governance
  • Benchmark: Reference only, not execution target

This framework keeps supplier roles explicit, controllable, and aligned with hyperscale procurement practice.

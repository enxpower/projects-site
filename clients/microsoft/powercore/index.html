W80P1 – Rack-Level Power Transfer Cord

External Specification Sheet (Official)

1. Project Overview

  • Project Code: W80P1
  • Product Name: Rack-Level Power Transfer Cord
  • Application: Internal power transfer within hyperscale data center racks (PDU to IT Equipment)
  • Service Environment: High-density, high-temperature, 24/7 continuous full-load operation
  • Positioning: Program-level execution supply (non-catalog item)

2. Electrical Specifications

ItemRequirementNotes
Rated Voltage250V ACSuitable for 208V / 240V rack environments
Rated Current15AFull-rated, non-derated
Frequency50 / 60 HzGlobal compatibility
Dielectric / Hi-PotPer UL requirementAs required by applicable UL standard

3. Connector Configuration

Connector A (Upstream / PDU Side)

  • Type: IEC 60320 C14 (Male)
  • Configuration: 3-prong, grounded

Connector B (Downstream / IT Equipment Side)

  • Type: IEC 60320 C15 (Female, High-Temperature)
  • Requirement: True high-temperature C15 housing (with notch)
  • Temperature Rating: ≥ 120 °C at connector head

4. Cable & Mechanical Construction

ItemRequirement
Conductor Size3 × 14 AWG
Conductor MaterialPure bare copper (No CCA, no mixed copper)
Cable TypeSJTW or UL-equivalent
Jacket Temperature Rating≥ 105 °C
Nominal Length5 ft (1525 mm)
Length Tolerance±30 mm
Measurement MethodEnd-to-end, excluding pins

5. Color Coding

  • A-Feed: Blue jacket
  • B-Feed: Red jacket
  • Consistency Requirement:
    • Solid, uniform color
    • No color mixing across batches

6. Compliance & Traceability

  • Certification: UL Listed (entire assembly covered under a single UL File)
  • UL File Number: Required (e.g. E115330 or equivalent)
  • Environmental Compliance: RoHS & REACH compliant

Jacket Marking (Mandatory)

  • UL identification mark
  • Manufacturer name or trademark
  • Wire gauge (14 AWG)
  • Voltage / current rating
  • Batch code or date code for traceability

7. Baseline & Options

  • Baseline Configuration: Non-locking
  • Optional (Future Discussion Only): Locking versions (not part of current baseline)

8. Vendor Evaluation Criteria

Vendors will be evaluated based on:

  • Compliance: Full alignment with C14–C15 / 14 AWG / SJTW / UL-listed configuration
  • Documentation: Valid UL File coverage and supporting documents
  • Consistency: Ability to maintain length and color tolerances in mass production
  • Traceability: Robust batch and lot tracking suitable for data center audits
  • Capacity: Stable, long-term production capability for rack-level programs

9. Version Control

  • Version: 1.1
  • Status: Baseline for program alignment
  • Change Policy: Architecture and ratings fixed; minor implementation details reviewed case by case

W80P1 – 机柜级电源传输线

对外规格书（正式版）

1. 项目概述

  • 项目代号： W80P1
  • 产品名称： 机柜级电源传输线
  • 应用场景： 超大规模数据中心机柜内部（PDU → IT 设备）电源传输
  • 运行环境： 高密度、高温、7×24 小时满载连续运行
  • 产品定位： 项目级执行物料（非目录标准品）

2. 电气规格

项目要求说明
额定电压250V AC适用于 208V / 240V 机柜环境
额定电流15A不降额
频率50 / 60 Hz全球通用
耐压 / Hi-Pot按 UL 要求依据适用 UL 标准

3. 接口配置

接口 A（上游 / PDU 侧）

  • 类型： IEC 60320 C14（公头）
  • 结构： 三芯接地

接口 B（下游 / 设备侧）

  • 类型： IEC 60320 C15（母头，高温型）
  • 要求： 真正高温 C15 外壳（带缺口）
  • 耐温等级： ≥ 120 °C（接插件头部）

4. 线缆与结构要求

项目要求
导体规格3 × 14 AWG
导体材质纯裸铜（禁止 CCA / 混铜）
线缆类型SJTW 或 UL 等效
护套耐温≥ 105 °C
标称长度5 英尺（1525 mm）
长度公差±30 mm
测量方式端到端测量，不含插针

5. 颜色区分

  • A 路电源： 蓝色护套
  • B 路电源： 红色护套
  • 一致性要求：
    • 单一纯色
    • 不同批次不得混色

6. 合规与追溯

  • 认证要求： UL Listed（整线组件必须在同一 UL File 覆盖范围内）
  • UL 文件号： 必须提供（如 E115330 或等效）
  • 环保合规： RoHS / REACH

护套印字要求（必须）

  • UL 标识
  • 制造商名称或商标
  • 线规（14 AWG）
  • 电压 / 电流等级
  • 批次号或日期码（用于追溯）

7. 基线与可选项

  • 当前基线： 非锁定型
  • 可选项（后续讨论）： 锁定型结构（非当前必选）

8. 供应商评估标准

供应商将基于以下维度进行评估：

  • 合规性： 是否完全符合 C14–C15 / 14 AWG / SJTW / UL Listed
  • 文件完整性： 是否能提供有效 UL File 及配套资料
  • 一致性： 是否能在量产中保持长度与颜色稳定
  • 追溯能力： 是否具备满足数据中心审计要求的批次管理
  • 产能保障： 是否具备长期、稳定的机柜项目供货能力

9. 版本管理

  • 版本号： 1.1
  • 状态： 项目基线版本（用于统一对齐）
  • 变更原则： 架构与额定参数固定，细节实现可逐项评审

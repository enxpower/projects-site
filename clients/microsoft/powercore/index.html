W80P1 Specs

W80P1 – External Specification Sheet

(For Power Cord Execution Vendors)

Project Code

W80P1 – Rack-Level Power Transfer Power Cord

Application

Internal power transfer within hyperscale data center racks,

typically between rack PDU and IT equipment.

This is a rack infrastructure component, not a retail power cord.

Electrical Specifications

  • Rated Voltage: 250V AC
  • Rated Current: 15A
  • Frequency: 50 / 60 Hz
  • Conductor: Copper
  • Conductor Size: 3 × 14 AWG

Connector Configuration

  • Connector A: IEC C14
  • Connector B: IEC C15 (true high-temperature type)

C15 must meet high-temperature requirements suitable for data center use.

Mechanical Specifications

  • Cable Length: 1525 mm ± 30 mm (5 ft nominal)
  • Cable Type: SJTW or equivalent (UL scope compliant)
  • Jacket Color:
    • Blue (A-feed)
    • Red (B-feed)

Compliance & Safety

  • UL Listed (UL File Number required)
  • Hi-Pot Test: 1500V
  • No breakdown / no short circuit
  • RoHS & REACH compliant

Marking & Traceability

  • UL marking on cable jacket
  • Manufacturer identification
  • Voltage / current rating
  • Batch or date code for traceability

Delivery & Commercial Notes

  • Supplied as rack-level BOM item
  • Stable mass production capability required
  • Non-locking configuration as baseline
  • Locking option may be discussed separately (optional, not mandatory)

Positioning Statement (Vendor Facing)

This product is intended as an execution-level supply for an existing hyperscale data center program.

The specification is fixed.

Vendors are evaluated on compliance, consistency, documentation, and delivery reliability.
